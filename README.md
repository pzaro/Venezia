# 🇮🇹 Venice Travel Guide (Based on Tasos Dousis)

Ένας συγκεντρωτικός, διαδραστικός οδηγός για τη Βενετία, βασισμένος στα tips από την εκπομπή "Εικόνες" του Τάσου Δούση, με επικαιροποιημένες τιμές (2025).

## 📋 Περιεχόμενα
Η λίστα περιλαμβάνει:
* **Αξιοθέατα** (San Marco, Rialto, Νησιά)
* **Φαγητό & Ποτό** (Bacari, Tramezzini, Εστιατόρια)
* **Κόστη** (Εισιτήρια, Μεταφορές, Τιμές φαγητού)
* **Google Maps Links** για άμεση πλοήγηση

## 🚀 Πώς να το χρησιμοποιήσετε
1. Κατεβάστε το αρχείο `index.html` και `locations.json`.
2. Ανοίξτε το `index.html` στον browser σας.
3. Πατήστε στα κουμπιά για οδηγίες πλοήγησης.

## 💡 Top Tips (Συνοπτικά)
* **Καφές:** Πίνουμε πάντα όρθιοι (**al banco**) για τιμή ~1.50€.
* **Μετακίνηση:** Συμφέρει η κάρτα **Vaporetto 48h** (35€).
* **Φαγητό:** Αποφύγετε τα μενού με φωτογραφίες ("Tourist Traps"). Προτιμήστε tramezzini στο *Bar Rialto da Lollo*.

## 🛠️ Τεχνικά
* Δεδομένα σε μορφή **JSON**.
* Frontend με απλή **HTML/JS**.
